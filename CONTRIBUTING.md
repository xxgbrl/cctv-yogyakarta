# Kontribusi

Terima kasih sudah tertarik berkontribusi! 🙏

## Cara Melaporkan Link CCTV yang Mati

Buka [Issues](../../issues) baru dengan format:
- **Judul:** `[Dead Link] Nama Kamera - Wilayah`
- **Isi:** URL stream yang bermasalah dan nama wilayahnya

## Cara Menambahkan Sumber CCTV Baru

1. Fork repo ini
2. Edit file `index.html` (tambah card wilayah baru) atau file `index.html` di dalam subfolder wilayah yang sesuai
3. Pastikan URL stream adalah sumber **publik resmi** (domain `.go.id` atau instansi resmi)
4. Buat Pull Request dengan deskripsi singkat sumber yang ditambahkan

## Aturan Kontribusi

- ✅ Hanya sumber CCTV publik dan resmi
- ✅ Tidak mengubah desain utama secara drastis
- ❌ Tidak menambahkan tracker, iklan, atau script pihak ketiga
- ❌ Tidak menambahkan CCTV privat atau tidak resmi
